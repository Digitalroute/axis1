Read Me

To check how the Hot Deployment work follow the following procedure.

Go to main class of ServiceTest then 
 1. Replace filename with a filename in your local machine
 2. Then inside the specified directory create two sub directory call
     "service" and "module"
 
 3. Create a service.xml using service1.xml file located inside the \test-resources\deployment
 4. Then take any .jar file and put that service.xml into the META_INF directory
 
 5. Then run the ServiceTest 
 6. While running the program put modified jar file into service director.
 


you can down load test jars from 
 http://www.apache.org/~deepal/HotDeploymentjars/
